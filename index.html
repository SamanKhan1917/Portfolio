Projects

Savorly

A personalized recipe recommendation system.

Savorly is a smart recipe platform powered by AI that transforms ingredients into personalized dishes. Utilizing OpenAI's LLM and NLP technologies, it suggests tailored recipes based on user inputs. Built with HTML, CSS, and JavaScript, it features a FastAPI and MySQL backend for efficient performance and seamless functionality.

HealthiFy

A food ordering website with chatbot support.

HealthiFy is an AI-powered platform that enhances healthy food ordering with a streamlined interface built using HTML, CSS, and JavaScript. It leverages AI and ML technologies, featuring a Dialogflow NLP-powered chatbot for personalized interactions, and utilizes a FastAPI and MySQL backend to ensure seamless, data-driven performance.

SereneTalk

A mental well-being companion chatbot.

SereneTalk is a smart mental health chatbot designed to provide support for individuals seeking comfort during difficult times. Powered by Rasa and utilizing Gemini LLM and NLP technologies, it engages users in empathetic conversations and offers coping strategies. Built with Streamlit for the user interface and leveraging technologies like Angular and Node.js, SereneTalk features a FastAPI and MySQL backend, ensuring efficient performance and a seamless user experience while prioritizing mental well-being.
